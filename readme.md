## Descrição do Projeto

Este projeto foi desenvolvido como atividade final da disciplina de Desenvolvimento Web. O objetivo é apresentar minha evolução ao longo do semestre, destacando os conteúdos aprendidos, os projetos realizados e meus objetivos para o futuro na área da tecnologia.

O site é composto por uma linha do tempo com os principais temas estudados durante as aulas, uma página sobre minha evolução no aprendizado de desenvolvimento web e uma página com meus próximos objetivos profissionais e acadêmicos.

Durante o desenvolvimento do projeto foram utilizados HTML5 e CSS3, aplicando conceitos como responsividade, Flexbox, Media Queries, animações e boas práticas de organização visual para criar uma interface moderna e adaptável a diferentes dispositivos.
